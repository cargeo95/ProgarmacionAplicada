Descripcin de la pgina y artculo a analizar

El propsito de este documento es realizar un anlisis detallado de los precios de productos en un sitio de comercio electrnico mediante tcnicas de scraping de datos. El scraping permite extraer informacin directamente de pginas web, facilitando la recoleccin de datos relevantes para la investigacin de mercado. Esta metodologa es especialmente til en el contexto del comercio electrnico, donde los precios y las ofertas cambian constantemente, y donde el anlisis de estos datos puede proporcionar informacin valiosa sobre la competitividad y la estructura de precios de una empresa.

Mediante el anlisis de los precios de productos especficos, es posible identificar patrones y tendencias, lo cual ayuda a comprender mejor la dinmica del mercado y las estrategias de precios utilizadas. Este tipo de anlisis es fundamental para evaluar la posicin de una empresa en el mercado y para detectar oportunidades de ajuste en sus estrategias comerciales y de precios, en beneficio de su competitividad.

Objetivos

Analizar la distribucin de precios de los productos en la pgina de comercio para identificar rangos de precio comunes y valores atpicos.

Evaluar la competencia en el mercado a travs de la comparacin de precios de productos similares en diferentes rangos.

Identificar posibles estrategias de precios de la empresa, comprendiendo cmo los productos se agrupan en ciertos intervalos de precio.

Determinar la existencia de patrones en los precios que sugieran diferencias en calidad, popularidad o estrategia de marketing.

Metodologa empleada de scraping

Herramientas y libreras utilizadas:

Se utiliz Python junto con libreras especficas para scraping, como requests, BeautifulSoup, y Selenium. Estas herramientas permitieron acceder y extraer informacin estructurada del sitio web de Like Me.

Proceso de scraping:

Acceso a la pgina: Se accedi a la seccin especfica de crop tops para mujeres. Extraccin de datos: Se recolectaron datos sobre el nombre del producto, el precio y el enlace de la imagen. Almacenamiento de datos: Los datos fueron almacenados en un DataFrame de Pandas para su posterior anlisis.

setup()

# Definir una lista para almacenar los resultados
scraped_data = []

# Define el Spider de Scrapy
class MiSpider(scrapy.Spider):
    name = "mi_spider"
    start_urls = ["https://www.likeme.com.co/mujer/crop-tops.html"]

    def parse(self, response):
      # Extrae los elementos deseados
      for product in response.css("div.product-item-info"):
          # Ajusta el selector CSS para extraer el nombre
          name = product.css("h2.h2-styles-for-items::text").get()
          if name:  # Asegrate de manejar casos donde el texto sea nulo
              name = name.strip()

          price = product.css("span.price::text").get()
          image = product.css("img.product-image-photo::attr(src)").get()

          # Guarda los datos en el array global
          scraped_data.append({
              'name': name,
              'price': price,
              'image': image
          })


# Define una funcin para ejecutar el Spider
@wait_for(20)  # Define un tiempo mximo de espera para el spider
def run_spider():
    # Limpiar los resultados anteriores
    scraped_data.clear()

    # Crear un runner para el Spider
    runner = CrawlerRunner()

    # Conectar una seal para saber cuando el crawler termina
    dispatcher.connect(lambda: reactor.stop(), signal=signals.spider_closed)

    # Ejecutar el spider
    d = runner.crawl(MiSpider)
    return d

# Ejecutar el spider
run_spider()

Grfico de Cajas (Box Plot):

La mediana se encuentra cerca de los 25,000, lo que confirma que la mayora de los precios estn alrededor de este valor. El rango intercuartlico (la distancia entre el primer y tercer cuartil) tambin es amplio, lo que sugiere una dispersin considerable en los precios.

Grfico de Violn:

La forma del grfico de violn es similar a la distribucin observada en el histograma, con una concentracin alta en la regin de 20,000 a 25,000. Tambin muestra simetra alrededor de la mediana, con una disminucin gradual hacia los valores extremos, aunque existen valores fuera de la concentracin principal.
